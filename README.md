# 😊 Sentiment Analysis Tool (NLP + Machine Learning)

An AI-powered Sentiment Analysis system that classifies text as **Positive** or **Negative** using Natural Language Processing and Machine Learning techniques.

Developed as part of an internship project at **Syntecxhub** 🚀

---

## 🚀 Project Overview

This project analyzes user input text and predicts sentiment using a trained ML model.  
It processes raw text, converts it into numerical features, and applies a Naive Bayes classifier for prediction.

---

## ✨ Features

- 🧹 Text preprocessing (cleaning & normalization)
- 🔤 TF-IDF Vectorization (uni-grams & bi-grams)
- 🤖 Naive Bayes classification model
- 📊 Model evaluation (Accuracy & F1-score)
- 💬 Real-time CLI-based sentiment prediction
- 😊 Confidence score with emoji output

---

## 🧠 Tech Stack

- Python 🐍  
- Pandas  
- Scikit-learn  
- NLP (Natural Language Processing)  
- TF-IDF Vectorizer  
- Naive Bayes Classifier
